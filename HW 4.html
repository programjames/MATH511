HW 4¶

We need $G'(x, \bar{x})$ to be a step function,

$$G'(x, \bar{x}) = \begin{cases}s & x < \bar{x}\\ s+1 & x \ge \bar{x}\end{cases}$$

for some value $s$. It should look like:

import matplotlib.pyplot as plt
import numpy as np
plt.rcParams.update({'font.size': 14})

x = np.linspace(0, 1, 1000)
y = np.zeros(x.shape)
x_bar = 0.7
s = 0.0
y[x < x_bar] = s
y[x >= x_bar] = s + 1

plt.plot(x, y)
plt.xlabel("x")
plt.ylabel(f"G'(x, {x_bar})")
plt.axvline(x_bar, color="red")
plt.text(x_bar-0.05,s+0.5,r"$\overline{x}$",rotation=90)
plt.show();

We have a Neumann boundary condition on the left, so $s=0$. Integrating again gives

$$G(x, \bar{x}) = \begin{cases}c&x< \bar{x}\\ c+(x-\bar{x})&x\ge \bar{x}\end{cases}$$

for some constant $c$. The Dirichlet boundary condition on the right gives

$$c + 1-\bar{x} = 0\Longleftrightarrow c = \bar{x} - 1.$$

So, we find

$$G(x, \bar{x}) = \begin{cases}\bar{x} - 1&x< \bar{x}\\ x-1&x\ge \bar{x}\end{cases}.$$

All that's left is to find $G_0(x)$ and $G_1(x)$. We try

$$G_0(x) = \lim_{\bar{x}\to 0}G(x, \bar{x}) = x-1$$

which indeed satisfies the conditions. For $G_1(x)$ we try

$$G_1(x) = \lim_{\bar{x}\to 1}G(x, \bar{x}) = 0$$

which again does satisfy the conditions (there's a mistake in the hw, it should say $u(1) = 0$).
